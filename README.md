# 🧪 React Practical Evaluation

## 📌 What You Are Building

You are working on a **Task Manager Application**.

This application should allow users to:

* Add new tasks
* Mark tasks as completed or pending
* View a list of tasks
* Filter tasks based on their status

The application is already partially built, but it is **not working correctly**.

1. Understand the existing code
2. Identify and fix issues
3. Improve and extend functionality

This test evaluates:

* Your understanding of React concepts
* Your debugging ability
* Your logical thinking

## 🐞 Part 1: Fix the Application

Make the application fully functional.

After fixing:

* You should be able to add tasks
* Toggle task completion correctly
* Ensure UI updates properly

### 🔍 Clues (Read Carefully)

* Input field may not behave as expected while typing
* Adding a task may not update the UI correctly
* Toggling a task might not reflect properly every time
* Filter options may show incorrect results
* List rendering might give warnings or unexpected behavior

👉 Carefully observe how state is updated and how values are passed.
